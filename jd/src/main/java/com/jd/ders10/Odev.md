# Ödev: Günlük Harcama Kontrolü

Bir kişinin günlük harcamalarını kontrol eden basit bir program yazmanız isteniyor.

--

Problem Tanımı

Kullanıcıdan aşağıdaki bilgileri alan bir program yazın:

- Günlük bütçe (örneğin: 100)
- Yapılan harcama miktarı (örneğin: 85)

Program, bu bilgilere göre aşağıdaki kuralları kontrol etmelidir:

Kurallar
- Eğer harcama bütçeden büyükse
→ "Bütçeyi aştınız!"
- Eğer harcama bütçeye eşitse
→ "Bütçenin tamamını kullandınız."
- Eğer harcama bütçeden küçükse:
    - Kalan para 20 veya daha az ise
→ "Dikkat! Bütçeniz bitmek üzere."
    - Kalan para 20’den fazlaysa
→ "Bütçeniz kontrol altında."
