# Ödev 2 – Orta Seviye (Switch–Case)
- Kullanıcıdan bir işlem türü seçmesi istenecektir (1: Toplama, 2: Çıkarma, 3: Çarpma, 4: Bölme).
- Kullanıcıdan iki adet sayı alınacaktır.
- Seçilen işlem türüne göre sonuç hesaplanacaktır.
- Bölme işleminde sıfıra bölme kontrolü yapılacaktır.
- Geçersiz işlem seçimlerinde hata mesajı verilecektir.
- Tüm işlem seçimleri switch–case yapısı ile gerçekleştirilecektir.
